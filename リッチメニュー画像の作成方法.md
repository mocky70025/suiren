# リッチメニュー画像の作成方法

## 重要

LINEのリッチメニューを表示するには、**画像が必須**です。画像がない場合、リッチメニューは表示されません。

## 画像の要件

- **サイズ**: 2500 x 1686 ピクセル
- **形式**: PNG
- **ファイル名**: `richmenu_image.png`
- **保存場所**: `assets/richmenu_image.png`

## 画像のレイアウト

リッチメニューは2つのエリアに分かれています：

1. **左側（0, 0, 1250, 1686）**: スタンプカードボタン
2. **右側（1250, 0, 1250, 1686）**: 支払いボタン

## 画像の作成方法

### 方法1: 画像編集ソフトを使用

1. 2500 x 1686 ピクセルの画像を作成
2. 左側に「スタンプカード」、右側に「支払い」のテキストやアイコンを配置
3. PNG形式で保存
4. `assets/richmenu_image.png` に配置

### 方法2: オンラインツールを使用

以下のようなオンラインツールを使用できます：

- Canva（https://www.canva.com/）
- Figma（https://www.figma.com/）
- Photoshop
- GIMP（無料）

### 方法3: シンプルな画像の例

以下のようなシンプルな画像でも動作します：

```
┌─────────────────┬─────────────────┐
│                 │                 │
│  スタンプカード  │     支払い      │
│                 │                 │
│                 │                 │
└─────────────────┴─────────────────┘
```

## 画像の配置

1. `assets` フォルダを作成（まだない場合）
2. `richmenu_image.png` を `assets/` フォルダに配置
3. GitHubにpush
4. Renderで再デプロイ

## 確認方法

1. サーバーを再起動（または再デプロイ）
2. ログを確認して「リッチメニュー画像をアップロードしました」と表示されることを確認
3. LINEアプリを再起動
4. 公式アカウントの下部にリッチメニューが表示されることを確認

## トラブルシューティング

### リッチメニューが表示されない場合

1. **画像が正しく配置されているか確認**
   - `assets/richmenu_image.png` が存在するか
   - 画像のサイズが 2500 x 1686 ピクセルか
   - 画像の形式がPNGか

2. **サーバーログを確認**
   - 「リッチメニュー画像をアップロードしました」と表示されているか
   - エラーメッセージがないか

3. **LINEアプリを再起動**
   - LINEアプリを完全に終了して再起動

4. **リッチメニューの状態を確認**
   - `/api/line/richmenu/status` エンドポイントで確認

## 注意事項

- 画像がない場合、リッチメニューは作成されません
- 画像のサイズが間違っている場合、アップロードに失敗する可能性があります
- LINEアプリを再起動しないと、リッチメニューが表示されない場合があります
