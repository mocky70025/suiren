# 運営用QRコード生成ページ

## アクセス方法

運営用ページは以下のURLでアクセスできます：

```
https://[あなたのサイトURL]/admin
```

例: `https://suiren.onrender.com/admin`

## 認証

運営用ページにはパスワード認証があります。

**デフォルトパスワード**: `admin123`

⚠️ **重要**: 本番環境では必ずパスワードを変更してください！

### パスワードの変更方法

`admin.js` ファイルの以下の部分を変更してください：

```javascript
const ADMIN_PASSWORD = 'admin123'; // ここを変更
```

変更後、GitHubにプッシュして再デプロイしてください。

## 使い方

1. `/admin` にアクセス
2. パスワードを入力してログイン
3. 「ユーザーIDを直接入力」を選択（または将来的に売り手一覧から選択）
4. 売り手のユーザーIDを入力
5. 「QRコードを生成」をクリック
6. 生成されたQRコードを印刷またはダウンロード
7. 売り手にQRコードを渡す

## 注意事項

- このページのURLは運営のみが知っている必要があります
- パスワードは定期的に変更することを推奨します
- セッションはブラウザを閉じると無効になります
