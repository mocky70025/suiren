# チャネルアクセストークンの取得方法

## 🎯 目的

LINE Messaging APIを使うために、チャネルアクセストークン（長期）を発行します。

## 📋 手順（詳しく）

### 1. LINE Developersにアクセス

1. ブラウザで [https://developers.line.biz/](https://developers.line.biz/) を開く
2. LINEアカウントでログイン

### 2. プロバイダーとチャネルを選択

1. ログイン後、作成したプロバイダー（例：すいれん）をクリック
2. 「Messaging API」チャネルをクリック
   - チャネルID: `2008552055` のチャネルです

### 3. Messaging APIページを開く

1. 左側のメニューから **「Messaging API」** をクリック
   - 「チャネル基本設定」の下にあります
   - アイコンは「💬」のような形です

### 4. チャネルアクセストークン（長期）を発行

1. ページを下にスクロール
2. **「チャネルアクセストークン（長期）」** というセクションを探す
   - 「チャネルアクセストークン（短期）」の下にあります
   - または、ページの下の方にあります

3. **「発行」** ボタンをクリック
   - ボタンは青い色です

4. 確認ダイアログが表示されたら **「発行」** をクリック

5. **トークンが表示されます**
   - 例：`xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
   - 長い文字列です（約200文字程度）

6. **このトークンをコピーして保存**
   - ⚠️ **重要**：このトークンは一度しか表示されません
   - コピーして安全な場所に保存してください

## 🔍 見つからない場合

### パターン1: 「Messaging API」メニューが見つからない

- 左側のメニューを確認
- 「チャネル基本設定」の下に「Messaging API」があります
- もしくは、チャネル設定ページの上部のタブで「Messaging API」タブをクリック

### パター2: 「チャネルアクセストークン（長期）」セクションが見つからない

- ページを下にスクロールしてください
- 「チャネルアクセストークン（短期）」の下にあります
- または、検索機能（Ctrl+F / Cmd+F）で「長期」と検索

### パターン3: 「発行」ボタンがない

- 既に発行済みの可能性があります
- 「再発行」ボタンがある場合は、それをクリック
- 既存のトークンが表示されている場合は、それをコピー

## 📸 画面のイメージ

```
LINE Developers
├─ プロバイダー名（すいれん）
│  └─ Messaging APIチャネル
│     ├─ チャネル基本設定
│     ├─ Messaging API ← ここをクリック
│     │  ├─ Webhook URL
│     │  ├─ チャネルアクセストークン（短期）
│     │  └─ チャネルアクセストークン（長期）← ここ
│     │     └─ [発行] ボタン
│     └─ ...
```

## ⚠️ 重要な注意事項

- チャネルアクセストークンは**秘密情報**です
- 一度発行すると、次回アクセスした時には表示されません
- 必ずコピーして保存してください
- このトークンがないと、LINE APIを使えません
