# LINE統合 ステップ一覧

## 📋 全ステップの概要

LINE公式アカウント統合を完了するための全ステップです。順番に実行してください。

### ✅ ステップ1: LINE Developersでアプリを作成
- LINE DevelopersでMessaging APIチャネルを作成
- チャネルアクセストークンとチャネルシークレットを取得
- 詳細：`LINE統合ステップ1.md`

### ✅ ステップ2: 環境変数の設定
- Renderやサーバーで環境変数を設定
- チャネルアクセストークン、チャネルシークレット、APP_URLを設定
- 詳細：`LINE統合ステップ2.md`

### ✅ ステップ3: パッケージのインストール
- `npm install`でLINE Messaging APIパッケージをインストール
- 変更をGitにコミット
- 詳細：`LINE統合ステップ3.md`

### ✅ ステップ4: Webhook URLの設定
- LINE DevelopersでWebhook URLを設定
- Webhookの利用を有効化
- 詳細：`LINE統合ステップ4.md`

### ✅ ステップ5: 動作確認とテスト
- LINE公式アカウントを友だち追加
- ポイントカード表示機能をテスト
- 詳細：`LINE統合ステップ5.md`

### 🔄 ステップ6: フロントエンド統合（今後実装）
- LINEログイン機能の追加
- 運営画面からのメッセージ送信機能
- 支払いリンク送信機能
