# PayPay IDの確認方法

## PayPay IDとは

PayPay IDは、PayPayアプリで個人間送金を行う際に使用する識別子です。
- 電話番号（例: 09012345678）
- PayPay ID（例: @paypay-id）

どちらでも送金できます。

## 確認方法

### 方法1: PayPayアプリで確認

1. PayPayアプリを開く
2. 右下の「マイページ」タブをタップ
3. 「PayPay ID」または「設定」をタップ
4. 「PayPay ID」を確認
   - 電話番号が表示される場合: その電話番号を使用
   - PayPay IDが表示される場合（例: @abc123）: そのIDを使用

### 方法2: 送金画面で確認

1. PayPayアプリを開く
2. 「送る」をタップ
3. 「QRコードを表示」をタップ
4. 表示されたQRコードの下にPayPay IDが表示されます

## 運営画面での入力

運営画面（`/admin`）で売り手のPayPay IDを入力する際は：

- **電話番号の場合**: `09012345678`（ハイフンなし）
- **PayPay IDの場合**: `mocky70025` や `@abc123` など（@マークは省略可）

例：
- `09012345678`（電話番号）
- `mocky70025`（PayPay ID）
- `@mocky70025`（@マーク付きでもOK、自動で除去されます）

## 注意事項

- PayPay IDは個人情報なので、適切に管理してください
- 電話番号とPayPay IDの両方が使えますが、どちらか一方を入力すればOKです
- 電話番号の方が一般的に使われやすいですが、PayPay IDでも問題なく動作します
