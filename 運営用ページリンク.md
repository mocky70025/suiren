# 運営用ページのリンク

## 🔗 運営用ページのURL

### ローカル開発環境の場合

```
http://localhost:3000/admin
```

### 本番環境（Render）の場合

```
https://あなたのRenderサービスのURL/admin
```

**例**：
- `https://suiren.onrender.com/admin`
- `https://your-app.onrender.com/admin`

## 🔐 認証情報

### パスワード

**デフォルトパスワード**: `admin123`

⚠️ **重要**: 本番環境では必ずパスワードを変更してください！

### パスワードの変更方法

1. `admin.js` ファイルを開く
2. 以下の部分を変更：

```javascript
const ADMIN_PASSWORD = 'admin123'; // ここを変更
```

3. 変更後、GitHubにプッシュして再デプロイ

## 📋 運営用ページでできること

1. **売り手情報の確認**
   - 累計入金金額
   - 取引回数
   - 取引履歴

2. **未処理の受け取り記録の処理**
   - 買い手名が入力されていない受け取り記録を処理
   - 買い手を選択してポイントカードに反映

3. **リッチメニューの管理**
   - LINE公式アカウントの下部メニューを作成・設定

4. **全データ一覧**
   - すべての受け取り記録を確認
   - すべての支払い記録を確認

## 💡 ブックマーク推奨

運営用ページは頻繁にアクセスするため、ブラウザのブックマークに追加することをおすすめします。

## ⚠️ セキュリティ注意事項

- このページのURLは運営のみが知っている必要があります
- パスワードは定期的に変更することを推奨します
- セッションはブラウザを閉じると無効になります
- 公共の場所では必ずログアウトしてください
